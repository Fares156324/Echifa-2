# EchifaVF template propre

Ce dossier contient une base nettoyee du site VETS a partir du miroir HTTrack interrompu.

## Template d'origine
- Template exact detecte: KnightOne
- Source du template: https://bootstrapmade.com/knight-simple-one-page-bootstrap-template/
- Le site VETS est une personnalisation importante de ce template.

## Dossier a utiliser
- Point d'entree principal: index.html
- Pages secondaires: about/index.html, become-partner/index.html, blogs/index.html, experts/index.html, blog-details.html
- Styles principaux: assets/css/main.css
- Scripts principaux: assets/js/main.js, assets/js/landing.js, assets/js/blog-details.js

## Nettoyage effectue
- Copie separee du miroir original
- Suppression du dossier recursif assets/img/assets
- Suppression des faux fichiers media en .html sous assets/img
- Correction de quelques references cassees dans les pages principales

## Limites restantes
- Le projet reste base sur un miroir, pas sur le depot source original.
- Certaines ressources partage/social ou illustrations d'origine peuvent manquer.
- Le blog depend encore de Contentful via JavaScript frontal.

## Conseils pour modifier le site
1. Commencer par index.html pour le contenu principal et la home page.
2. Adapter les textes/images dans assets/img et les pages secondaires.
3. Verifier ensuite les blogs si tu veux garder ou supprimer la dependance Contentful.
4. Tester localement toutes les pages avant de publier.
